RottenDouban v5.0 - 烂番茄豆瓣聚合评分
==========================================

核心功能:
- 烂番茄新鲜度 + 爆米花指数展示
- 豆瓣评分 + 热门短评展示
- 加权评分 (影评人0.3 + 观众0.3 + 豆瓣0.4)
- 纯API数据获取，无浏览器依赖
- GitHub Actions 自动部署
- 暗色/亮色主题，响应式布局

数据源:
- 烂番茄: Algolia 搜索 API
- 豆瓣: 搜索 API + 缓存

部署:
- 推送到 GitHub，Actions 自动获取数据并部署到 Pages
- 本地无需运行爬虫
